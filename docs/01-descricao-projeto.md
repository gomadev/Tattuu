# 01 - Descrição do Projeto

## Nome e Contexto

**Projeto:** Tatt-oo Data Platform  
**Contexto de negócio:** plataforma digital para estúdios e tatuadores gerenciarem agenda, orçamento, atendimento e análise operacional.

## Problema

Estúdios de tatuagem geralmente operam com dados fragmentados em planilhas, mensagens e sistemas isolados. Isso dificulta:

- previsão de demanda;
- gestão de agenda e no-show;
- análise de receita por profissional e estilo;
- tomada de decisão com base em dados confiáveis.

## Objetivos

- Centralizar dados operacionais e eventos em uma arquitetura de dados escalável.
- Disponibilizar dados confiáveis para análises táticas e estratégicas.
- Estruturar base para evoluir para um protótipo completo na Parte 2.

## Stakeholders

- Dono(a) do estúdio
- Tatuadores
- Time financeiro/administrativo
- Time de marketing
- Analistas (consumo de dashboards)
