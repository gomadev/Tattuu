# 06 - Considerações Finais

## Riscos e Limitações

- Complexidade de múltiplos componentes em máquina local.
- Limitações de hardware para cargas mais altas.
- Dependência de dados sintéticos na fase inicial.
- Tempo curto para implementação completa no semestre.

## Próximos Passos (Parte 2)

1. Subir stack mínima em Docker (backend, frontend, banco, observabilidade básica).
2. Implementar ingestão de eventos reais da aplicação web.
3. Criar pipeline batch diário para camada Silver.
4. Publicar indicadores na camada Gold para dashboards.
5. Definir testes de qualidade e monitoramento de pipelines.
